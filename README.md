# Vanilla UJS

It is implementation of Rails jQuery UJS in pure JavaScript. Based on [code of ≈Åukasz Niemier](http://github.com/hauleth/vanilla-ujs).
No extra dependencies.

## Installation

The vanilla-ujs files will be added to the asset pipeline and available for you to use. Just add these lines in `app/assets/javascripts/application.js`:

```js
//= require vanilla-ujs
```
